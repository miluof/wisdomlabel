AI标注工具 wisdomlabel 
1，支持多种图形的标注，点，矩形，多边形，样条等。
2，更加人性化的操作方式。
2.1支持鼠标中键滚动放大缩小图形。
2.2修改矩形支持4条边和5个点修改。
2.3对频繁操作的指令支持快捷键快速操作，节省不少时间
2.4支持默认标签的设定，对于只标注某一种标签的标注，节省不少时间。
3，更加丰富的功能
3.1支持标签批量替换。
3.2支持标签数量统计。
3.3支持镜像，旋转图形和改变图形的大小，标签跟随变化。
3.4支持抠图与抠图粘贴，带标签一起粘贴。
3.5支持图片批量重命名，标签文件跟随变化。
3.6支持图片名模糊搜索
3.7支持根据已标注，未标注已经标签名称选择图片
3.8支持图片另存。
3.9支持视频转图片
3.10在清洗大量连续图片时，支持一次跳过多帧。
3.11在清洗大量图片时，支持设定临时目标并将目标复制到临时目录。
3.12支持移走没有标注的图片。
3.13支持移走没有图片的标注。
3.14支持删掉图片上不清晰或者干扰的区域。
